# 🔥 NitroVerse

یک پلتفرم بازی آموزشی با تم فضایی و رقابتی

## 🚀 اجرا

فقط کافیه **`start.bat`** رو دابل‌کلیک کنی!

## 🎮 چطور بازی کنم؟

1. ثبت‌نام کن
2. وارد شو
3. شروع به بازی کن
4. امتیاز جمع کن و رنک‌ت رو بالا ببر

## 🛠️ تکنولوژی‌ها

- Flask (Python)
- SQLite
- HTML/CSS/JS
